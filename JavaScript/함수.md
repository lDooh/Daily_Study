# 자바스크립트 함수

## 함수(function)

+ 하나의 특별한 목적의 작업을 수행하도록 설계된 독립적인 블록
+ 자바스크립트에서는 함수도 하나의 타입(datatype)이다.
  + 따라서 함수를 변수에 대입하거나, 함수에 프로퍼티를 지정하는 것도 가능하다.

## 함수 정의

```js
function function_name(parameter1, parameter2, ...) {
  ...
  
  return ...;
}
```
